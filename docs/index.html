mnml.css

A minimal, configurable CSS design system

The goal of mnml.css is to be a minimum-viable design system. Its features include:

  • A CSS reset compatible with modern browsers
  • A mobile-first, responsive design approach
  • A preference for semantic tags over classes, when appropriate
  • Easy-to-implement customization using CSS Custom Properties
  • Dark-mode support

Configuration

mnml.css uses CSS Custom Properties throughout to allow you to customize sizing, spacing, colors, and animation timing.
